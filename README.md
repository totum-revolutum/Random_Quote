  Основні вимоги:

Використовувати Vue.js для інтерфейсу. Також можна користуватися Vue CLI або Vite.

Після завантаження сторінки одразу відобразити випадкову цитату.

Наявна кнопка для генерації нової цитати без перезавантаження сторінки.

Поруч з цитатою відобразити ім'я її автора.

Відобразити user-friendly повідомлення у випадку помилки роботи із сервером.

Використовувати Git під час розробки.


  Бонусом буде:

Відображення історії згенерованих цитат у вигляді списку.

Можливість скопіювати цитату в буфер обміну (clipboard) в один клік.

Можливість налаштування деяких параметрів генерації випадкових цитат.

Можливість поширити цитату за допомогою соцмереж (Telegram, Facebook ...).
